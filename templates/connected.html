¡Bienvenido, tu conexión con Beetrack ha sido exitosa!


¡Ya solo falta el ultimo pazo para que completes tu conexión entre tus cuentas de Beetrack y Shopify!

Para finalizar tu conexión debes de configurar un webhook dentro de tu cuenta de Beetrack. Para ello, redirígete a login.beetrack.com e inicia sesión con tus credenciales Beetrack.
Una vez que hayas entrado a tu cuenta beetrack, dirígete a Configuraciones -> Configuración avanzada -> Webhooks -> Crear nuevo webhook (botón con el signo “+”).
Para configurar el webhook utiliza el Url y los Headers que se presentan a continuación.
Esto permitirá que tu cuenta de Beetrack se comunique en tiempo real con tu cuenta de Shopify.

Url = beetrack.shopify.com
Headers = {
Account Uuid : {{content}}
Contetn Type : Application/json
}
